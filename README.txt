This is the full version of the Each Tweet Counts project.
http://eachtweetcounts.com/

It requires the tweepy library.
https://github.com/tweepy/tweepy/

And, that you get your own access tokens.
https://dev.twitter.com

Unicode information.
https://dev.twitter.com/docs/counting-characters

::WIKI::

Single letters that cannot be tweeted:
D, F, G, L, M, S, W (there may be others)

Two character combinations:
/n

Single letter commands:
@username message
#topic
D username message (direct message)
M username message (private message)
F username (follow)
L username (leave, stop following)
G username (get most recent tweet)

SMS Commands:
http://support.twitter.com/articles/14020-twitter-sms-commands

Other discoveries:
1. Any tweet with spaces preceding characters will have spaces stripped.
2. Any duplicate tweet (might need to be consecutive) will not be tweeted. 

Let me know what you find.
